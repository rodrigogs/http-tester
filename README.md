# http-tester

Simple library and created in a hurry to test and log HTTP requests.

## How to install?
Have node.js installed
> npm install http-tester --g

## How to use?
> http-test /my/file/name http://my-url.com 2000 5000

Args: 
* 1: log file name
* 2: url to request
* 3: request timeout
* 4: request interval
